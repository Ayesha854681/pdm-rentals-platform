# PDM Rentals Platform

## Project Overview

PDM Rentals is a full-stack online vehicle rental platform developed to simplify vehicle booking and rental management. The system allows users to browse vehicles from different categories, submit bookings, and manage rental information through a database-driven backend.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

---

## Features

- User-friendly vehicle rental interface
- Multiple vehicle categories
- Booking system with customer information
- Automatic rental cost calculation
- Database connectivity using PHP and MySQL
- Server-side form handling
- ID document upload functionality
- Category-based search system
- Responsive website design

---

## Vehicle Categories

- Luxury Sedans
- SUVs
- Convertibles
- Electric Cars
- Superbikes
- Choppers
- Sports Cars
